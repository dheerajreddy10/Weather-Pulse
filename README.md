# 🌤️ Weather Pulse

Weather Pulse is a modern weather forecasting application built using **JavaScript**, **Tailwind CSS**, and the **OpenWeatherMap API**. The application provides real-time weather updates, hourly forecasts, and 5-day weather predictions through a clean and responsive user interface.

## 🚀 Features

* 🔍 Search weather by city name
* 🌡️ Real-time temperature and weather conditions
* 💧 Humidity information
* 🌬️ Wind speed monitoring
* 📊 Atmospheric pressure display
* 👁️ Visibility information
* 🌅 Sunrise time
* 🌇 Sunset time
* ⏰ Hourly weather forecast
* 📅 5-Day weather forecast
* 📱 Responsive design for different screen sizes

## 🛠️ Technologies Used

* HTML5
* JavaScript (ES6+)
* Tailwind CSS
* OpenWeatherMap API

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/dheerajreddy10/Weather-Pulse.git
```

### Navigate to Project Folder

```bash
cd Weather-Pulse
```

### Install Dependencies

```bash
npm install
```

## 🔑 API Configuration

Create an account at OpenWeatherMap and generate an API key.

Replace the API key in your JavaScript file:

```javascript
const APIKEY = "YOUR_API_KEY";
```

## ▶️ Running the Application

Open the `index.html` file in your browser.

Or use Live Server in VS Code:

1. Install the Live Server extension.
2. Right-click `index.html`.
3. Select **Open with Live Server**.
